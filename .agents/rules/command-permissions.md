# Global Command Permissions

- **Authorized Commands**: The following command suites are authorized to be run with `SafeToAutoRun: true` in all contexts for this project:
    - `npm`
    - `npx`
    - `git`

- **Context**: These commands are considered safe and standard for the development workflow of this application.
